# r35krag0th's Public Dotfiles

## Steps

- Install [asdf](https://asdf-vm.com) &rarr; Homebrew or [instructions](https://asdf-vm.com/guide/getting-started.html#_3-install-asdf)
- Install [pyenv](https://github.com/pyenv/pyenv) &rarr; [instructions](https://github.com/pyenv/pyenv#homebrew-in-macos)
- Install [pyenv-virtualenv](https://github.com/pyenv/pyenv-virtualenv) &rarr; Use homebrew or [instructions](https://github.com/pyenv/pyenv-virtualenv#installation)
- Install [neovim](https://neovim.io) (via homebrew)
- Install [vim-plug](https://github.com/junegunn/vim-plug) &rarr; [instructions](https://github.com/junegunn/vim-plug#unix-linux)
- Install [krew](https://krew.sigs.k8s.io) &rarr; [instructions](https://krew.sigs.k8s.io/docs/user-guide/setup/install/)
